# Ely-Tesia — MIDI Pack 01

Arreglos MIDI base y simplificados para probar el piano roll, el teclado virtual,
el mapeo de rango y la reproducción.

## Especificación

- Standard MIDI File Type 1
- 480 PPQ
- Pista 0: tempo y metadatos
- Pista 1: melodía
- Pista 2: acompañamiento
- General MIDI Acoustic Grand Piano
- Canal 1: melodía
- Canal 2: acompañamiento

## Canciones

1. Happy Birthday to You
2. Campanita / Twinkle, Twinkle, Little Star
3. La Bamba
4. Korobeiniki
5. Bella Ciao
6. Morning Mood
7. La primavera — Vivaldi
8. El Cóndor Pasa

## Nota

Son bases de demostración, no transcripciones de concierto completas.
El Cóndor Pasa queda marcado para revisión territorial antes de una
distribución global.
